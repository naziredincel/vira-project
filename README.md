 🚗 Vira - Kurumsal Araç Filo Yönetim Sistemi

Vira, araç kiralama sektöründeki operasyonel süreçleri dijitalleştiren bir Java masaüstü uygulamasıdır.

 Özellikler

- Araç yönetimi (Car, SUV, ElectricCar)
- Müşteri yönetimi
- Rezervasyon yönetimi ve çakışma kontrolü
- Veriler dosyaya kalıcı olarak kaydedilir

 Kullanılan Teknolojiler

- Java (OOP)
- Java Swing (Arayüz)
- Dosya sistemi (Veri saklama)

 Nasıl Çalıştırılır?

1. Projeyi ZIP olarak indirin veya klonlayın
2. Eclipse'te **File → Import → General → Existing Projects into Workspace** seçin
3. İndirilen klasörü seçin
4. `ui/Main.java` dosyasını açın
5. **Run As → Java Application** ile çalıştırın
